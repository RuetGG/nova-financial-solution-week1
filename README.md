# News Analysis Project

This repository contains the analysis for Task-1 of the weekly news dataset. The goal was to explore headlines, publication patterns, text content, and publisher contributions.

## What Was Done

* **Headline Statistics:** Calculated lengths, averages, and percentiles.
* **Publisher Analysis:** Identified most active publishers and analyzed email domains.
* **Publication Trends:** Examined daily, weekly, yearly, and hourly article patterns.
* **Text Analysis:** Extracted common keywords and phrases to highlight major topics.
* **Time Series Insights:** Observed spikes in article volume corresponding to market events.

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/RuetGG/nova-financial-solution-week1.git
```

2. Switch to the `task-1` branch:

```bash
git checkout task-1
```

3. Open notebooks to view analysis and visualizations.

This project provides an overview of news publishing behavior, topic prevalence, and organizational contributions.
